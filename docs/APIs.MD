# API Endpoints

**Base URL:** `VITE_API_URL`

---

| Resource      | Endpoint        | Description                  |
| ------------- | --------------- | ---------------------------- |
| Posts         | `/posts`        | List all posts               |
| Post Detail   | `/posts/{id}`   | Details of a specific post   |
| Authors       | `/authors`      | List all authors             |
| Author Detail | `/authors/{id}` | Details of a specific author |
| Categories    | `/categories`   | List all categories          |
